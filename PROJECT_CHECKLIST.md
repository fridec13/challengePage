# Challenge! 프로젝트 개발 체크리스트

## 📋 프로젝트 개요
- **목적**: 친구들과 함께하는 범용 챌린지 관리 플랫폼
- **기술스택**: React + TypeScript + Tailwind CSS + Supabase + Vercel
- **특징**: 모바일 우선 반응형 디자인, Pretendard 폰트
